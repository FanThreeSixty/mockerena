currency
========

.. code-block:: python

    fake.cryptocurrency()
    # ('ZCL', 'Zclassic')

    fake.cryptocurrency_code()
    # 'ZCL'

    fake.cryptocurrency_name()
    # 'BlackCoin'

    fake.currency()
    # ('BMD', 'Bermudian dollar')

    fake.currency_code()
    # 'BMD'

    fake.currency_name()
    # 'Haitian gourde'
